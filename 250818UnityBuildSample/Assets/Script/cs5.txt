빌드 설정

Player Setting 에서 설정하는 값

1. CompanyName : 레지스트리 저장소
2. Product Name : 프로젝트 / 게임 / 앱의 이름
3. Version : 현재의 파일 버전

Icon : 플랫폼 별 아이콘을 설정하는 기능
		모바일 작업에서는 필수적으로 진행해줍니다.

Cursor : 프로그램 내부에서 사용할 커서의 이미지
		-> 기본 이미지만 설정 가능

Resolution & Presentation 설정
Run In Background
해당 설정을 킬 경우, 유니티 화면에서 벗어나도 실행 됩니다.

FullScreen Mode : 해상도 설정

Standalone 기준 설정
(유니티에서 빌드 할수 있는 PC 환경용 앱)
 ㄴ Windows, macOS, Linux 환경

1. 플레이어 로그 파일 생성 여부
2. 창 크기 조절 기능
3. 게임 창 포커스를 잃어도 렌더링 할지에 대한 여부
4. Alt + Enter로 전체화면 기능을 제공할지에 대한 여부
5. 멀티 플레이 환경에서 첫번째 화면만 사용할지 여부
6. 고급 랜더링 설정 여부

Splash Image
이미지 필요

로고 :  화면에 보이는 메인 이미지
백그라운드 : 배경이미지

에디터에서는 Priview를 통해 기능 체크 가능

로고 기능에서 show Unity Logo를 제거하면 유니티 로고가 나오지 않음
단 로고 리스트에 로고가 추가 되어있어야합니다

로고와 백그라운드로 사용하는 이미지는 Sprite 설정을 해줘야 사용 가능합니다.

배경 설정 : 배경 위에 덮이는 오버레이 투명도 설정
블러 : 뒷 배경을 흐리게 설정
Alternate Portrait Image : 세로 방향에서 실행되는 앱일 경우 대체 이미지로써 사용할 수 있습니다.
